# MITE

![Mite Logo](Logo/logo.png)


MITE stands for

```
Mite
Item
Time
Emit
```

each of which in turn stand for:
```
Item
Time
Emit
Mite
```

```
Time
Item
Mite
Emit
```

```
Emit
Mite
Item
Time
```

... and so on.

MITE is a recursive acronym generator.

# Usage
```bash
> python MITE.py 5

ORGAN

```
